# Goal: Competitive Analysis
Performed a competitive analysis on a product, application or service based on heuristics.

# Tasks Completed

- Selected a competing product, application or service to analyze.

- Created a list of heuristics to use for your evaluation.

- Evaluated the design based on your list of heuristics.

- Analyzed what I like about the design, why it works and where it breaks down.

- Identified opportunities and weaknesses to improve design.

- Documented key findings and takeaways from the analysis.

- Documented Design Progress
